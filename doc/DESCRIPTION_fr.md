Médiathèque est une **collection personnelle de médias** en ligne : films, envies, notes, enrichissement TMDB, import/export CSV, questionnaire du soir, prêts entre amis, et stockage de fichiers volumineux (magazines, etc.).

Version incluse : **0.3.0** (moteur [Moncine](https://github.com/daryl40000/mediatheque)).
